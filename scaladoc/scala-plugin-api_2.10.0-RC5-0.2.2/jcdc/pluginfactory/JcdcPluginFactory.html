jcdc.pluginfactory

JcdcPluginFactory

class JcdcPluginFactory extends JavaPlugin

This plugin and this code is not intended for use. It is just an empty plugin that is required for turning this library into a plugin, so that the API and Scala can be on the classpath for plugins that want to use this API.
